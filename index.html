                Out of the Box
            


            


                

                    
It uses Emmet!


                    
Emmet is a popular markup and stylesheet builder. Using Emmet's conventions, it becomes just much easier to use.

                    
Plus, between developers, there's a singular style guide!

                


                

                    .block { d:b } => .block { display: block; }
                

            


            


                

                    
Breakpoints


                    
Wyldstyle prioritizes mobile first design. 

                    
With one look at your markup, you can immediately see which styles are for other device sizes.

                


                

                    
                        <h1 class="brand [ u-fz:3r@l ]"></h1>
                    
                

            


            


                

                    
CLI Watcher


                    
You can watch specific files to parse the utility classes on.

                


                

                    wyldstyle --watch
                

            


            


                

                    
Configurability


                    
It can easily be configured to suit your workflow.

                    
You can use direct values, variables, or even mixins for your breakpoint configuration. 

                    
Write your own custom snippets that will make your workflow much faster!

                


                

                    
{
    breakpoints: {
        m:  breakpoint(tablet),
        l:  1024px,
        xl: $breakpoint-retina
    },
    directories: [index.html],
    output: css/utilities.css,
    emmet: {
        snippets: {
            m:0a: margin: 0 auto
        }
    }
}
